
Getting and Cleaning Data Course Final peer graded Project

This repository is a submission for Getting and Cleaning Data course project. The instructions show how to run analysis on the Human Activity recognition dataset.

Dataset
Human Activity Recognition Using Smartphones

Files
CodeBook.md a code book that describes the variables, the data, and any transformations or work that I performed to clean up the data

run_analysis.R prepares the data and then followed by the 5 steps for the course project:
Merges the training and the test sets and create one data set.
Extracts only measurements on mean and standard deviation for each measurement.
Use descriptive activity names to name the activities in the data set
label the data set with descriptive variable names.
from the data in step 4, create a 2nd, independent set with the average of each variable for each activity and each subject.
Q5.txt is the exported final data set.
